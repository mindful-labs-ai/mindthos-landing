## 메타 타이틀

### 형식 및 길이
- **길이:** 30-60자 (한국어)
- **핵심 키워드:** 앞쪽 30자 이내에 배치
- **브랜드명:** 파이프(|)로 구분하여 뒤에 추가 (선택)

### 권장 패턴
```
번아웃 증상과 회복 방법: 직장인이 알아야 할 5가지 | 앤아더라이프
우울증과 우울감의 차이: 전문가가 설명하는 구분법
부부 갈등 해결을 위한 상담 가이드 | 앤아더라이프
```

### 금지 사항
- 키워드 나열형 제목 ("우울증 불안 스트레스 상담")
- 클릭베이트 표현 ("충격적인", "당신은 모르는")
- 60자 초과 (검색 결과에서 잘림)

### 형식 및 길이
- **길이:** 120-155자 (한국어)
- **구조:** 핵심 내용 요약 + 주요 키워드 + 행동 유도

### 권장 패턴
```

정보로 도와드립니다.
```

### 포함 요소 체크리스트
- [ ] 주요 키워드 자연스럽게 포함
- [ ] 글의 핵심 가치 제안
- [ ] 행동 유도 문구 (마지막 문장)
- [ ] 120자 이상, 155자 이하

## 헤딩 구조 (H1-H3)

### H1 (글 제목)
- 반드시 1개만
- 주요 키워드 포함 필수
- 독자의 검색 의도를 직접 반영

### H2 (주요 섹션)
- **최소 2개의 H2에 주요 또는 보조 키워드 포함**
- 자연스러운 문장 형태 권장 (질문형 가능)
- 5-8개 사용 (글 길이에 따라 조정)
- 예시:
  - "번아웃의 주요 증상 5가지"
  - "번아웃 회복에 도움이 되는 상담 방법"
  - "직장인 스트레스 관리, 혼자 해결하기 어려운 이유"

### H3 (하위 섹션)
- H2 내용을 세분화할 때만 사용
- 키워드 포함은 선택 사항
- 남용하지 않음 (H2 당 최대 3개)

## 도입부 (첫 문단) SEO 규칙

- **주요 키워드를 첫 100자 이내에 포함**
- 독자의 검색 의도와 직결되는 내용으로 시작
- 스니펫(Featured Snippet) 획득을 위해 직접적인 답변 가능성 고려
- 글의 전체 내용을 예고하는 "로드맵" 문장 포함

## 이미지 SEO

### alt 텍스트 작성 기준
- 이미지 내용을 구체적으로 설명
- 주요 또는 보조 키워드 자연스럽게 포함
- 40자 이내로 간결하게
- 장식용 이미지: `alt=""` (빈 값)

### 예시
```
alt="번아웃을 경험하는 직장인이 피로한 표정으로 책상에 앉아있는 모습"
alt="부부 상담 세션에서 상담사와 대화하는 커플"
alt="심리상담 공간의 편안한 소파와 식물 인테리어"
```

### 파일명 규칙
- 한국어 사용 금지
- 영문 소문자, 하이픈 구분
- 내용 설명적인 이름
- 예: `burnout-recovery-counseling.jpg`, `couple-therapy-session.jpg`

## 내부 링크 전략

### 글당 내부 링크 수
- **권장:** 2-4개
- **최소:** 2개 (상담 프로그램 또는 관련 블로그 글)
- **최대:** 5개 (과도한 내부 링크는 사용자 경험 저하)

### 링크 유형별 배분
| 링크 유형 | 개수 | 비고 |
|---------|-----|-----|
| 관련 블로그 글 | 1-2개 | 같은 카테고리 또는 연관 주제 |
| 상담/교육 프로그램 페이지 | 1-2개 | CTA 역할 |
| 카테고리 목록 페이지 | 0-1개 | 필요 시 |

### 앵커 텍스트 작성 기준
- 링크 목적지 내용을 명확하게 설명
- "여기를 클릭", "이 글" 등 범용 텍스트 금지
- 키워드 포함 자연스럽게

## 외부 링크

### 글당 외부 링크 수
- **권장:** 1-3개
- 신뢰할 수 있는 출처만 연결

### 허용 출처
1. 보건복지부, 국립정신건강센터
2. 한국심리학회, 한국상담학회
3. WHO, APA (American Psychological Association)
4. PubMed 등재 학술 논문
5. 서울대, 연세대 등 주요 대학 연구 자료

### 외부 링크 속성
```html
<a href="https://..." target="_blank" rel="noopener noreferrer">링크 텍스트</a>
```
- `target="_blank"`: 새 탭에서 열기
- `rel="noopener noreferrer"`: 보안 속성 필수

## URL 슬러그

### 작성 규칙
- 영문 소문자만 사용
- 단어 구분: 하이픈(`-`)
- **3-5단어** 권장
- 의미 없는 단어 제거 (관사, 전치사 등)
- 한국어 음역 사용 가능 (예: `burnout-hoebok` 보다 `burnout-recovery` 선호)

### 예시
```
/blog/burnout-symptoms-recovery
/blog/couple-counseling-guide
/blog/youth-anxiety-counseling
/blog/counseling-first-session
/blog/depression-vs-sadness
```

## 구조화 데이터 (Schema Markup)

### 필수 스키마 (모든 블로그 글)
1. **BlogPosting Schema**
   - `headline`: 글 제목
   - `description`: 메타 디스크립션
   - `author`: 작성자 정보
   - `datePublished`: 발행일
   - `dateModified`: 수정일
   - `image`: 대표 이미지 URL

2. **FAQPage Schema**
   - FAQ 섹션이 있는 모든 글에 적용
   - 4-6개 질문-답변 쌍

### 선택 스키마 (해당 시 적용)
- **BreadcrumbList**: 상위 카테고리 > 글 제목 경로
- **Person**: 작성자가 특정 상담사인 경우
- **Organization**: 센터 소개 관련 글
