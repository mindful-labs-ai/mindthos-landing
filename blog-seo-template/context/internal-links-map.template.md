## 블로그 포스트 내부 링크 규칙

### 글당 내부 링크 구성 (2-4개)

```
[필수] 1개: 해당 카테고리 CTA 목적지 (상담 또는 교육 프로그램 페이지)
[권장] 1-2개: 같은 카테고리 또는 연관 주제 블로그 글
[선택] 0-1개: 상위 카테고리 목록 또는 다른 관련 프로그램 페이지
```

### 링크 배치 위치

| 위치 | 링크 유형 | 비고 |
|-----|---------|-----|
| 본문 중간 (자연스러운 맥락) | 관련 블로그 글 | 문장 흐름 방해하지 않게 |
| 섹션 말미 | 심화 정보 블로그 글 | "더 자세히 알고 싶다면~" |
| 본문 마지막 문단 | CTA 목적지 (프로그램/예약) | 행동 유도 |
| InlineCTA 컴포넌트 | 상담 프로그램 | 글 중간 또는 끝 |
| BottomCTA 컴포넌트 | 상담/교육 프로그램 | 글 하단 자동 삽입 |

## 필러-클러스터 내부 링킹 전략

### 필러 페이지 (Pillar Pages)
각 카테고리의 가장 포괄적인 핵심 글:

| 카테고리 | 필러 페이지 주제 예시 |
|---------|-----------------|
| mental-health | "마음 건강 완벽 가이드: 증상부터 치료까지" |
| counseling-stories | "심리상담 A to Z: 처음 시작하는 분을 위한 안내" |
| relationships-communication | "건강한 관계를 위한 소통 완전 가이드" |
| children-youth | "아이 마음 건강 가이드: 부모가 알아야 할 모든 것" |
| self-growth | "심리학으로 보는 자기 성장 완전 가이드" |
| expert-column | "상담사를 위한 임상 실무 완전 가이드" |
| education-certification | "심리상담사 자격증 완벽 가이드" |

### 클러스터 링킹 규칙
- 클러스터 글 → 필러 페이지: 항상 링크 (앵커: "○○에 대해 더 자세히 알아보기")
- 필러 페이지 → 클러스터 글: 관련 세부 주제로 연결
- 클러스터 글 ↔ 클러스터 글: 같은 카테고리 내 관련 글 상호 링크

## 카테고리별 연관 블로그 글 연결 패턴

### mental-health
```
우울증 증상 글 ←→ 우울증 치료 방법 글
번아웃 증상 글 ←→ 번아웃 회복 방법 글
불안장애 글 → 불안 해소 실용 팁 글
자존감 낮은 증상 글 ←→ 자존감 회복 방법 글
```

### counseling-stories
```
상담 비용 안내 글 ←→ 상담 효과 글
첫 상담 준비 글 → 상담사 선택 방법 글
상담 과정 글 ←→ 상담 종결 후기 글
```

### relationships-communication
```
부부 갈등 원인 글 ←→ 부부 상담 효과 글
의사소통 기술 글 → 비폭력 대화 실습 글
이혼 고민 글 → 부부 상담 가이드 글
```

### children-youth
```
청소년 우울증 글 ←→ 부모 대처법 글
아동 ADHD 글 ←→ 가정에서 돕는 방법 글
학교 부적응 글 → 청소년 상담 안내 글
```

### self-growth
```
자기 이해 글 ←→ 자기 수용 방법 글
마음챙김 입문 글 ←→ 일상 명상 실습 글
완벽주의 글 ←→ 자기 비판 극복 글
```

### expert-column + education-certification
```
상담 기법 이론 글 ←→ 실제 적용 사례 글
자격증 안내 글 ←→ 수련 과정 경험담 글
슈퍼비전 방법 글 → 사례 개념화 글
```

## 전환 퍼널별 내부 링크 흐름

### 내담자 전환 퍼널
```
블로그 글 (정보형 키워드 유입)
    ↓ InlineCTA / 본문 내 링크
/counseling 또는 /counseling/[slug]  ← 상담 프로그램 상세
    ↓ "상담 예약하기" 버튼
/contact  ← 상담 예약 폼 작성
    ↓
예약 완료 (전환)
```

### 전문가 전환 퍼널
```
블로그 글 (전문가 키워드 유입)
    ↓ InlineCTA / 본문 내 링크
/programs  ← 교육 프로그램 목록
    ↓ /programs/[slug]
    ↓ "수강 신청하기" 버튼
/contact  ← 교육 문의 폼 작성
    ↓
수강 등록 (전환)
```

## 금지 사항

- 같은 문장에서 동일한 목적지로 2개 이상의 링크 사용 금지
- 관련성 없는 글을 강제로 연결하는 링크 금지
- "클릭하세요", "여기를 누르세요" 등 비설명적 앵커 텍스트 금지
- 한 글에 `/contact` 직접 링크 2개 초과 금지 (CTA 컴포넌트가 담당)
- 외부 페이지와 같은 앵커 텍스트 혼용 금지
