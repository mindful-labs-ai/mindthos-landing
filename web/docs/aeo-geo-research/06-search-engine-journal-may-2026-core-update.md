## (1) 2026-05 Core Update

- **시작**: 2026-05-21
- **롤아웃 윈도우**: 최대 2주
- **공식 코멘트**: 없음 (블로그 포스트, 목표 명시 전부 부재)
- **업계 추정**: 이번 업데이트가 **"AI citation을 위해 과최적화한 사이트"** 를 타깃할 수 있다는 의견 다수
- 시사점: AEO/GEO hacks (chunking, 인위적 mention 빌딩, FAQ 스팸 등)에 의존한 사이트가 변동 위험. → 공식 가이드 메세지와 일치

## (2) Google I/O 2026 Search Overhaul

- > "Biggest upgrade to the Search box in over 25 years." (Google)
- **Gemini 3.5 Flash** = AI Mode 기본 모델
- **Dynamic search box**: multimodal input (이미지, 파일, 음성)
- **Information agents**: 2026 여름 출시, 웹 변화를 지속 모니터링·요약
- **AI Mode 월간 US 사용자 10억 돌파**, 쿼리는 분기마다 더블링
- 사용자 행태 변화: **conversational, longer 쿼리** + follow-up 쿼리 **월 +40%**

## (3) llms.txt — 신호 충돌

| 출처 | 메시지 |
|------|--------|
| Google Search team | "**llms.txt는 AI Search에 불필요**" |
| Lighthouse 13.3 | 파일 없으면 기본으로 경고 표시 |
| John Mueller | "Markdown 문서는 **agentic 도구**에 도움. 하지만 현재 SEO 우선" |

해석:
- **검색 노출 목적으로는 llms.txt 만들 이유 없음** (Google 공식)
- **에이전트/MCP/AI 클라이언트 친화** 목적이라면 보조 자료로 의미 있음
- Lighthouse 경고는 추천일 뿐 — 따라가지 않아도 페널티 없음

## 마음토스 시사점

1. **5/21 이후 GSC 변동성 모니터링 필수** — 트래픽 변동이 우리 콘텐츠 품질 문제인지 코어 업데이트 영향인지 구분.
2. AI Mode 사용자 10억 → 한국 진입은 시차 있지만 **conversational long 쿼리** 비중 증가 가속 가능. "심리상담 비교" 같은 짧은 키워드보다 "**ADHD 같은데 일에 집중이 안되는 30대 직장인 검사 어디**" 같은 풀 문장 대응 필요.
3. **llms.txt 현재 미적용 유지**가 합리적. 한국어 AI 클라이언트 시장 분명해지면 그때 재검토.
4. follow-up 쿼리 +40% → fan-out에 대응되는 **passage-level 컨텐츠** (Digital Applied 가이드 #3 항목)가 곧 핵심.
