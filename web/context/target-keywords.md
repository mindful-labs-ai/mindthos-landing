# 마음토스 타겟 키워드 클러스터

마음토스 블로그의 주 독자는 **상담사 (임상가 / 수련생 / 슈퍼바이저 / 개업 상담사)** 입니다. 일반 대중 대상 "OO 증상" 키워드는 직접 타겟팅하지 않고, 상담 실무·운영·자기돌봄 관점의 키워드를 우선합니다.

## 검색 의도 분류

| 의도 유형 | 설명 | 예시 |
|---------|-----|-----|
| 정보형 (Informational) | 임상 기법·이론·절차 학습 | "사례개념화 방법", "작업동맹 측정" |
| 상업형 (Commercial) | 도구·교육·자격 비교 | "축어록 자동화", "상담사 교육 비교" |
| 거래형 (Transactional) | 즉시 행동 | "마음토스 가입", "축어록 무료 체험" |
| 탐색형 (Navigational) | 브랜드 검색 | "마음토스 가격", "Mindthos 보안" |

## 키워드 사용 규칙

1. 검색량이 명시되지 않은 경우 본 가이드는 **방향성 제시용**. 실제 발행 시 `python3 scripts/seo-analysis/keyword_research.py "<키워드>" --serp` 로 DataForSEO 실 데이터 확인 후 결정.
2. 카테고리별 핵심 키워드는 **필러 페이지** 후보. 한 카테고리에 필러 글 1편 + 클러스터 5-10편 구조 권장.
3. 일반 대중 키워드 ("우울증 증상" 등) 는 마음토스 블로그의 주 타겟이 아니므로 다루지 않음. 일반 대중 콘텐츠는 마음토스의 자매 사이트(앤아더라이프) 가 담당.
